## ⚙️ Функції квізу

- **65 питань** у форматі НМТ (4 варіанти відповіді)
- **Таймер 30 секунд** на кожне питання
- Пояснення після кожної відповіді
- Перегляд помилок із правильними відповідями в кінці
- Питання перемішуються при кожному запуску
